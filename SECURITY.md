# Security Policy

## Supported Versions

Only the latest version of Server Doctor is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.7.x   | :white_check_mark: |
| < 1.7   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Server Doctor, please do NOT open a public issue. Instead, please report it privately by emailing hassansalah120120@gmail.com.

We take security seriously and will investigate all reports promptly. We will aim to respond within 48 hours and provide a fix as soon as possible.

### What to Include in Your Report

- A detailed description of the vulnerability.
- Steps to reproduce the issue (proof of concept).
- Potential impact and severity.
- Any suggested mitigations.

Thank you for helping keep Server Doctor secure!
